#### 介绍

该项目是基于 Vue 和 vite 搭建的一个 PC 端的个人博客网站，其中融合了高德地图，网易云音乐等，Echarts 图表等一系列功能。

#### 项目安装

```
//首先你应当克隆该项目
git clone https://github.com/zxy1816378898/zxyWind.git

// 然后 安装项目依赖
npm install

// 接下来 部署服务端
a) 克隆服务端项目并运行

// 运行项目
npm run dev
```
